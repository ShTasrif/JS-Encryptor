# JS-Encryptor
Marshall encryptor for python script 
## Install on Termux :
```
pkg update -y && pkg upgrade -y
pkg install python -y
pkg install git -y
git clone https://github.com/ShTasrif/SH-Mail
```

# Run Termux:
```
cd JS-Encryptor
python jsenc.py
```
